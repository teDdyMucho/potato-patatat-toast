**Date:** 2026-05-28

## ✅ What we shipped

### 1. Admin dashboard (all 5 modules) — `/admin`
Full PRD + tracker in [docs/ADMIN_PAGE.md](./ADMIN_PAGE.md). Gated by a Supabase `profiles.role`.
- **Foundations** — `profiles` table + RLS, server-side admin guard (`lib/admin.ts`), admin shell (sidebar + top bar), "Admins only" 403 screen, role-gated **Admin** link in the nav.
- **Users & access** — list all users (service-role), promote/demote, suspend/reactivate (also bans in Supabase Auth), self-protection.
- **AI tool usage** — `tool_usage` table, `logToolUsage()` hooked into the tool routes, usage dashboard (totals, per-tool, top users, recent activity).
- **Content / Blog** — `blog_posts` table, full CRUD, manager + editor UI, **cover-image upload** (Supabase Storage), **article body** field, public `/blog` reads from DB (static fallback), **blog article pages** at `/blog/[slug]`.
- **Leads** — `leads` table, public `POST /api/leads`, the **contact form now persists** (it previously discarded submissions), admin leads inbox (status + notes + delete).
- Overview KPIs (users, sessions today, open leads, content items) are all live.

### 2. Public site UI/UX
- **Nav header** — removed Book Call from the header (kept in footer/CTAs), user **name → dropdown** with Log out, pinned flush-right, improved spacing.
- **Contact form** — dark input fields (were white), on-palette chips, sending/error states.
- **AI Tools page** — futuristic teal-grid backdrop behind the cards; **animated filter pills** (sliding teal indicator, hover/entrance motion).
- **Design Adjuster** — full futuristic redesign (HUD panels, corner brackets, scan-line) + a rich **generating/loading animation** while waiting for output, suggestion chips, download/regenerate.
- **About → Process** — steps now **reveal on scroll** as animated HUD panels (sweep + glowing node).
- **Home** — toned-down 3D logo rotation + canvas fill so the wordmark stays centered; WebGL crash fixed with graceful fallback.

### 3. Integrations
- **Chatbot → n8n webhook** — the home chat now proxies to `…/webhook/aktservices-chatbot` (server-side, flexible reply parsing, session id for memory).
- **GoHighLevel affiliate badge** — top-right hero badge ("Try GoHighLevel now") with pulsing ripple + animated arrow → opens the affiliate link.

### 4. Reliability fixes
- **WebGL context leak** — added `forceContextLoss()` + try/catch fallback in the 3D logo and GLSL hills (was crashing `/about` after navigation).
- 2D **"AKT" fallback** renders when the browser can't provide WebGL.
